# Contributing

## Dependencies

- [Yarn](https://yarnpkg.com/en/)

```
$ npm install -g yarn
```

## Setup

```
$ git clone git@github.com:JordanAdams/devdocs-cli.git
$ yarn install
```

## Testing

```
$ yarn test
```

or

```
$ yarn test:watch
```

## Linting

```
$ yarn lint
```

## Building

```
$ yarn build
```
