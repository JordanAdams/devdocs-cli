# Installation

```
$ npm i -g devdocs-cli
```

# Usage

Either search for both docs and pages with:

```
$ devdocs

  Search Docs: 
  
  Search Pages: 
```

or if you already know the docs slug, use:

```
$ devdocs <lang-slug>

  Search Pages:
```
